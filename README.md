## Dev tools
### compiledb
Dealing with python on my os proved to be quite annoying, so I let [pyenv](https://github.com/pyenv/pyenv) handle that instead.

This will set the user's python version to 3.13, you could also use the `pyenv local` command instead.
```bash
pyenv install 3.13
pyenv global 3.13
```


Then you can use pip, which will now be pointing at python 3.13 to install compiledb.
```bash
pip install compiledb
```
